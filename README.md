# Autional Developers

开发者站点，服务于 Autional SDK、Quickstart、Migration 与 API Reference 等面向集成开发者的内容展示。

基于 [Astro](https://astro.build) 构建，部署目标为静态站点。

## Development

```bash
pnpm install
pnpm dev
pnpm build
```

默认本地端口：`4433`
